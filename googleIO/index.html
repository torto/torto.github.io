O que é essa PIRA?

Node.js é um plataforma baseada no runtime JS do Chrome, para aplicações escaláveis em rede. Node.js é baseado em eventos, seu modelo é no-blocking I/O. Oque torna ele leve e eficinete, perfeito para aplicações em tempo real e de muitos dados ..

Como funciona hj?

  • Cliente faz uma requisição
  • Servidor atende e processa a requisição
  • Servidor devolve a resposta para o cliente
  • Pronto para processar uma nova requisição

Como é no Node?

  • Cliente faz uma requisição
  • Servidor passa a solicitação para processamento.
  • Pronto para processar uma nova requisição
  • Servidor lida com mais requisições a medida que ele vão sendo requisitados
  • Servidor fornece a resposta para o cliente quando a requisição concluiu o processamento.
